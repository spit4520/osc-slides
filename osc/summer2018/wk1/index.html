C# Goes To/Is Approached By Operator

April 2010, C# language designers put in a new, revolutionary operator without affecting the language's release schedule.
This is the operator '-->' 

int x = 1000, a = 0;
// this is read "while x goes to zero"
while (x --> 0)
{
	if (x % 3 == 0 || x % 5 == 0) {
		a += x;
	}
}
Console.WriteLine(a);

233168

int x = 1000, a = 0;
// this is read "while zero is approached by x"
while (0 <-- x)
{
	if (x % 3 == 0 || x % 5 == 0) {
		a += x;
	}
}
Console.WriteLine(a);
